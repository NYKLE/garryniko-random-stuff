# ProgressCircle

Round indicators for HP/MP, timers, counters, etc.

Clockwise and anti-clockwise fill. Starting angle by degrees, 0 is up. Line thickness adjustment by outer and inner clip diameter. Sections and progress jumping section by section option. Sections spacing is percentage of section width.
